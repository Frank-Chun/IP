📕 专利法（重点考条线路图）

以下为“考试角度”归纳，不是法规全文替代；法条原文请以 CNIPA / 国务院公开文本为准。

§2 发明的定义（技术方案）

  • 关键词：产品 / 方法 / 其改进 / 新的技术方案
  • 🎯 考点：实用新型必须是有形产品（形状/构造或其结合），不保护方法
  • 🚫 典型干扰：智力活动的规则和方法（商业方法本身）→ §25 + §2 双杀

§5 不授予专利权（公序良俗）

  • “违反法律、社会公德或者妨害公共利益”→ 不是“用途可能违法就毙”，而是客体/方案本身性质
  • 典型：专用于赌博的设备 / 伪造货币的设备 / 人胚胎工业应用等

§6 职务发明（三类情形）

  • ① 执行本单位任务（本职 / 特别委派 / 离职1年内与原任务相关）
  • ② 主要利用单位物质技术条件 + 约定未另定 → 归单位
  • 🎯 考法：给一小段事实，让你判定申请权归属 + 发明人署名权（署名权不可转让/继承）

§22 授权三性（新颖性·创造性·实用性）

  • 新颖性：单独对比；抵触申请（他人/本人的他案，但必须是“向CNIPA提出+申请日前申请、公开在申请日/优先权日之后”）
  • 创造性：最接近现有技术→区别→实际解决的技术问题→技术启示（反向考法常问“哪个不能推出显而易见”）
  • 实用性：产业上可制造/使用 + 积极效果（永动机/瞬间分解物/纯抽象=no）

§25 不授权主题（背熟）

  • 科学发现 / 智力规则 / 疾病诊断治疗方法 / 动物/植物品种（但微生物品系可） / 原子核变换方法
  • 🎯 考法：表面像“技术”，骨子里是“信息/生物本体/抽象规则”→ §25 毙掉

§26 申请文件 + 公开充分 + 支持/清楚（实务生死线）

  • 说明书：清楚完整，能够实现（§26.3）
  • 权利要求书：得到说明书支持 + 清楚（§26.4）
  • 发明人/申请人/代理人手续、修改不得超原说明书和权利要求书记载范围（§33）

§29/§30 优先权（外国12月 / 本国12月）

  • 优先权日→当作“新起点”来判断现有技术
  • 外观：外国6个月 / 本国6个月（设计专用）

§34/§35 公布 + 临时保护 + 实审请求（3年内）

  • 申请日后18个月公布；可提前公布
  • 实审必须请求+缴费，3年内不请求=视为撤回
